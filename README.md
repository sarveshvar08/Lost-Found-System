## 🚀 About the Project

A modern desktop application built using Python that helps users **report, search, and manage lost items efficiently**.  
It features a clean UI, image support, and an easy-to-use system for tracking lost belongings.

## ✨ Features

- 🔍 Search items quickly  
- 📋 View all items in a modern UI  
- ➕ Report lost items  
- 📸 Upload & preview images  
- 📍 Location tagging  
- 📞 Contact information  
- ✅ Checkout (mark item as claimed)  
- 🎨 Modern dark UI (CustomTkinter)
